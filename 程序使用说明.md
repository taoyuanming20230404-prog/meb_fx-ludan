# fx_ludan.exe 使用说明（免安装版）

本说明适用于通过 **`build_windows.bat`** 打包得到的 **`fx_ludan.exe`**。  
使用 exe **不需要安装 Python**。

---

## 一、使用前请确认

| 项目 | 说明 |
|------|------|
| **Windows 电脑** | 建议 Windows 10/11。 |
| **Google Chrome** | 必须已安装，且能正常打开网页。 |
| **ChromeDriver** | 版本需与 Chrome **主版本号**一致（如 Chrome 131 对应 chromedriver 131）。 |

**ChromeDriver（按优先级，任选其一即可）：**

1. **推荐**：将 `chromedriver.exe` 放在 **`fx_ludan.exe` 同一文件夹**（程序会默认使用）；  
2. 设置环境变量 **`FX_CHROMEDRIVER`** = `chromedriver.exe` 的完整路径（会覆盖同目录）；  
3. 把 `chromedriver.exe` 放到系统 **PATH** 中（仅当以上都未配置时由 Selenium 自动找）。

> 若启动后提示无法启动浏览器或连接失败，多半是 Chrome 与 ChromeDriver **主版本号**不一致。

---

## 二、文件放置

1. 将 **`fx_ludan.exe`** 放在任意文件夹（建议单独建一个文件夹，便于存放输出文件）。  
2. **可选**：在 exe **同一目录**下放置：  
   - **`std_keywords.txt`** — 标准项目词（每行一个词）  
   - **`synonyms.json`** — 同义词映射（JSON）  

若同目录有上述文件，程序会**优先使用**，覆盖 exe 内自带的词库；便于不改程序、只改文本即可调整匹配。

---

## 三、Excel 要求

准备一个 **`.xlsx`** 文件，第一行为表头，至少包含：

| 列名 | 必填 | 说明 |
|------|:----:|------|
| **号码** | 是 | 客户手机号；为空则该行跳过。 |
| **项目** | 建议填 | 用于匹配下拉项目并填写说明。 |
| **城市** | 建议填 | 用于选择地区。 |
| **微信** | 否 | 有则填。 |

---

## 四、操作步骤

1. **双击运行** `fx_ludan.exe`（首次可能略慢，属正常）。  
2. 程序会**自动打开 Chrome** 并打开录单相关页面。  
3. 在浏览器里**完成登录**（按你们后台要求操作）。  
4. 登录完成后，回到 **黑色命令行窗口**，按 **回车** 继续。  
5. 在弹出的窗口中**选择要录单的 Excel 文件**。  
6. 等待程序自动逐行处理；窗口中会显示 **进度** 和每行 **状态**。  
7. 全部结束后可关闭窗口。

> 若当前版本固定为「流程一」，将先查重再录单；具体以程序内提示为准。

---

## 五、运行后生成的文件

| 内容 | 位置 |
|------|------|
| **录单反馈表** | 与您选择的 Excel **同一文件夹**，文件名形如：`录单反馈+日期.xlsx`，多一列 **「状态」**（已录单 / 重单 / 跳过 / 失败）。 |
| **重单记录** | 在 **exe 所在目录**（及当月子文件夹内），文件名形如：`重单记录_年-月-日.txt`；同一天多次运行会自动加 **`_2`、`_3`** 等后缀，避免覆盖。 |
| **月份文件夹** | 如 `2026-03`，用于存放当月重单记录副本。 |

## 六、常见问题

**1. 双击没反应或一闪而过**  
- 尝试在 exe 所在文件夹空白处 **Shift+右键 →「在此处打开 PowerShell/命令提示符」**，输入 `.\fx_ludan.exe` 回车，查看报错信息。

**2. 提示无法连接浏览器 / 拒绝连接**  
- 关闭所有 Chrome 后重试；检查 ChromeDriver 是否与 Chrome 版本一致。

**3. 杀毒软件拦截**  
- 将 `fx_ludan.exe` 加入信任区，或联系网管放行。

**4. 控制台里出现英文报错（GCM、DEPRECATED 等）**  
- 多为 Chrome 自身日志，一般**不影响录单**，可忽略。

**5. 想改项目匹配规则**  
- 在 exe 同目录放好 **`std_keywords.txt`** 和 **`synonyms.json`** 后重新运行即可。
